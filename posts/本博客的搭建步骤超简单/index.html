本博客的搭建步骤（超简单）#

下面的教程都是用mac的环境进行，linux和windows平台大家可以照着改巴改巴

安装hugo#

brew install hugo

创建一个项目#

hugo new site my-project-name --format yaml

拉取PaperMod主题文件#

cd my-project-name
git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1

检查文件结构#

目前的文件目录结构应该是这样

.
├── archetypes
│   └── default.md
├── assets
├── content
├── data
├── hugo.yaml
├── i18n
├── layouts
├── static
└── themes
    └── PaperMod

配置主题#

echo "theme: PaperMod" >> hugo.yaml

运行命令#

hugo server

访问#

访问：http://localhost:1313/
